# Dependency Visualizer

Инструмент для визуализации графа зависимостей пакетов Rust.

## Этап 1: Минимальный прототип ✅

- Загрузка конфигурации из CSV файла
- Обработка ошибок параметров  
- Вывод параметров в формате ключ-значение

## Этап 2: Сбор данных ✅

- Получение Cargo.toml из GitHub репозиториев
- Парсинг зависимостей из секций [dependencies]
- Вывод прямых зависимостей пакета
- Обработка ошибок сети и парсинга

## Этап 3: Основные операции ✅

- Построение полного графа зависимостей алгоритмом DFS
- Обработка циклических зависимостей
- Режим тестирования с файловым репозиторием
- Рекурсивный обход транзитивных зависимостей

## Этап 4: Дополнительные операции ✅

- Поиск обратных зависимостей (reverse dependencies)
- Построение обратного графа зависимостей
- Поиск прямых и транзитивных обратных зависимостей
- Вывод пакетов, которые зависят от заданного пакета
